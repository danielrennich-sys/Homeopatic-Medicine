AI Repertorization Agent

The AI agent uses Claude (by Anthropic) to intelligently analyze your intake form responses. It understands homeopathic context — for example, it knows "worse from running" is a modality, not "running nose."

How to get an API key:
1. Go to console.anthropic.com
2. Create an account or sign in
3. Go to API Keys in the sidebar
4. Click Create Key and copy it

Cost: Each intake analysis costs ~$0.003 (one third of a penny). Anthropic gives $5 free credits on signup (~1,500 searches).

Privacy: Your API key is stored only in your browser's local storage. It is sent directly to Anthropic's API — never to any other server. The intake form data is sent to Claude for analysis and is not stored by Anthropic per their API data policy.

Without an API Key

The database search and intake form work without an API key using keyword-based matching. The AI agent adds contextual understanding but is not required. You can always use the full database search at the search page.
